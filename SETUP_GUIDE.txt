----------------------------

STEP 1: COPY PROJECT FILES
--------------------------
1. Copy the entire "chatbot_project" folder
2. Paste it into your XAMPP's htdocs folder:
   
   Windows:   C:\xampp\htdocs\chatbot_project\
   Mac:       /Applications/XAMPP/htdocs/chatbot_project/
   Linux:     /opt/lampp/htdocs/chatbot_project/


STEP 2: START XAMPP SERVICES
----------------------------
1. Open XAMPP Control Panel
2. Click "Start" for Apache
3. Click "Start" for MySQL
4. Wait until both show "Running" status


STEP 3: CREATE DATABASE
-----------------------
1. Open browser and go to: http://localhost/phpmyadmin
2. Click "New" on left sidebar
3. Enter database name: ai_chatbot_db
4. Select collation: utf8mb4_general_ci
5. Click "Create"


STEP 4: IMPORT DATABASE SCHEMA
------------------------------
1. Click on "ai_chatbot_db" database (left sidebar)
2. Click "SQL" tab at the top
3. Open the file: database/schema.sql (use Notepad or any text editor)
4. Copy ALL the SQL code from the file
5. Paste it into the SQL box in phpMyAdmin
6. Click "Go" button
7. You should see "4 tables created" and "4 rows inserted" messages


STEP 5: GET YOUR GROQ API KEY
-----------------------------
1. Visit: https://console.groq.com/
2. Sign up for a free account
3. Verify your email
4. Click "API Keys" in the dashboard
5. Click "Create API Key"
6. Give it a name (e.g., "My Chatbot")
7. Click "Create"
8. Copy the API key (it starts with "gsk_")
   âš ï¸ IMPORTANT: Save this key somewhere safe!


STEP 6: CONFIGURE API KEY
-------------------------
1. Open the file: config/database.php
2. Find this line:
   define('GROQ_API_KEY', 'YOUR_GROQ_API_KEY_HERE');
3. Replace YOUR_GROQ_API_KEY_HERE with your actual API key
4. Example:
   define('GROQ_API_KEY', 'gsk_abc123xyz456...');
5. Save the file


STEP 7: TEST THE APPLICATION
----------------------------
1. Open your browser
2. Go to: http://localhost/chatbot_project/
3. You should see the login page

   Username: admin
   Password: admin123

ğŸ› COMMON PROBLEMS & SOLUTIONS
-------------------------------

Problem: Can't access localhost
Solution: Make sure Apache is running in XAMPP

Problem: Database connection failed
Solution: 
  - Check MySQL is running in XAMPP
  - Verify database name is "ai_chatbot_db"
  - Check username is "root" and password is empty

Problem: API error when chatting
Solution:
  - Check your Groq API key is correct
  - Make sure you have internet connection
  - Verify the key in config/database.php

Problem: Page shows errors
Solution:
  - Make sure all files are in htdocs/chatbot_project/
  - Check PHP version (needs 7.4+)
  - Look at XAMPP error logs for details

Problem: Can't login
Solution:
  - Clear browser cookies
  - Try creating a new account
  - Check if database was imported correctly
